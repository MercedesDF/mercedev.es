# Documento de Intención · mercedev.es (v1.0)

## 1. Qué es esta web
Esta web es una biblioteca profesional viva donde documento cómo construyo, refuerzo y reutilizo sistemas técnicos. No es un portfolio estético ni un blog personal al uso. Es un espacio de trabajo público donde el conocimiento se organiza, se conecta y evoluciona.

## 2. Para quién es
- Equipos técnicos y de ingeniería
- Empresas con sistemas complejos
- Personas que valoran claridad, seguridad y mantenibilidad
- Reclutadores que buscan criterio, no solo listas de tecnologías

## 3. Qué problema resuelve
Permite mostrar pensamiento estructural y criterio profesional: no solo qué hago, sino cómo pienso, por qué tomo decisiones y qué aprendo cuando algo falla.

## 4. Qué NO es esta web
No es un escaparate de diseño, ni un repositorio de certificados, ni una colección de modas tecnológicas, ni una marca personal basada en eslóganes, ni una tienda online convencional.

## 5. Principios (Art de Coté)
Art de Coté es crear de forma colateral: aprovechar lo descartado, recomponer lo que se rompe y reforzar el sistema para que la siguiente iteración sea más sólida. El error se documenta. El proceso se explica. La estética sirve a la comprensión, no al revés.

## 6. Tono y forma
Tono claro, sobrio y honesto. Se prioriza legibilidad, orden y espacio.

## 7. Rol de Mercí
Mercí es un ayudante simbólico y contextual: la conciencia ligera del sistema. Observa y señala criterios sin interferir ni juzgar.

## 8. Evolución
Esta web no se considera terminada. Está pensada para crecer con proyectos, aprendizajes y herramientas propias.
