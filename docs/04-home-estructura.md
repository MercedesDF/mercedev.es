# 04-home-estructura.md · Arquitectura de Home (v0)

## Objetivo
Definir la estructura mínima de la Home antes de implementar HTML/CSS.

## Secciones
1. Propuesta de valor (qué hago + desde dónde pienso)
2. Marco Biblioteca (metáfora de estantería)
3. Recorrido evolutivo (obra civil -> procesos -> web -> laboratorio)
4. Accesos directos (Bitácora, Laboratorio, Sobre mí)
5. Presencia sutil de Mercí (no interactiva)

## Reglas
- Sin-JS usable.
- Jerarquía plana, máximo 2 niveles.
- Accesibilidad WCAG AA.
